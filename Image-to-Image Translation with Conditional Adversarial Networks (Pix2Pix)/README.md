### Download Facades Dataset

```bash
wget -N http://efrosgans.eecs.berkeley.edu/pix2pix/datasets/facades.tar.gz -O ./data/facades.tar.gz
```

```bash
tar -zxvf ./data/facades.tar.gz
```
